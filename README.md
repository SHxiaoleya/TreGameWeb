# 🎮 TreGameWeb

一个用于打发时间的开源 H5 小游戏仓库。  
所有游戏均可在浏览器中直接运行，无需安装，支持在线体验。

---

## ✨ 项目目标

- 收集和开发轻量、有趣的 H5 小游戏
- 保持代码简单易读，适合学习与二次开发
- 提供在线演示，打开即玩

---

## 🌐 在线演示

👉 演示地址：`https://你的用户名.github.io/你的仓库名/`

> 建议使用最新版 Chrome / Edge / Safari 访问。

---

## 🕹 当前已上传游戏

|---|---|---|
